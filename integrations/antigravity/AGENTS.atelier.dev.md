# Atelier — Antigravity Default Identity

This file is staged for Antigravity-specific workspace guidance when a lightweight host-specific entrypoint is needed alongside the root `AGENTS.md`.

Use Atelier MCP tools first for file I/O, search, edits, and shell work. If an Atelier MCP tool returns `noop`, is hidden, or is unavailable, use Antigravity or `agy` native reads/search plus shell `rg` or `grep`.
