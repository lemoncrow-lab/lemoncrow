# Milestone v1.0: Atelier Code Intelligence

**Status:** ✅ SHIPPED 2026-05-23
**Phases:** 1–7
**Total Plans:** 21

## Overview

This milestone delivered the full M0–M18 code-intelligence program for Atelier. The system routes agent queries by shape (name, pattern, semantic, text, history, deleted, cross-language) through a `SymbolIntelStore` composite with content-addressed cache, token-budget packing, and always-on trace. All work lands on existing MCP tool surfaces — no new top-level tool registrations.

**Core value shipped:** Agents can find and change code through budget-aware, precomputed intelligence with near-zero token overhead by default.

## Milestone Summary

**Key Decisions:**

- [Phase 01] Use repo-local `.atelier/cache/scip/<repo_id>/*.scip` artifacts with local-only binary discovery for M1 bootstrap path.
- [Phase 01] Default code search to `snippet=none` so hardened symbol lookup stays budget-safe unless callers opt into snippets.
- [Phase 02] Resolve ast-grep availability through env override, exact binary discovery, then pinned managed bootstrap before returning `tool_unavailable`.
- [Phase 02] Use the file's exact line-span text rather than the dedented symbol payload when applying symbol replacements through rich edit.
- [Phase 03] Keep semantic and hybrid ranking on the existing `code op="search"` surface with an additive `mode` parameter.
- [Phase 04] Pin pygit2 exactly at 1.19.2 and gate git-history code behind `require_pygit2()`.
- [Phase 05] Ratified option-a: Zoekt standalone serves search workloads only.
- [Phase 05] Keep M17 to literal-only static cross-language edges surfaced additively.
- [Phase 06] Keep bootstrap implicit on `tool_get_context` by enqueueing one deduped job on the existing worker path.
- [Phase 06] Reject external dependency symbol edits in `symbol_edit.py` before any file read.
- [Phase 06] Keep multi-repo support in a workspace parser/router helper layer instead of making `CodeContextEngine` multi-root.

**Issues Resolved:**

- Context overflow on code-intel tool calls — solved with token-budget packing.
- Stale SCIP index not detected until mutation — solved with `index_sha` gating.
- External dep symbols surfacing in workspace results — solved with `scope="external"` opt-in routing.

**Issues Deferred:**

- DEFR-01: Broader cross-language/runtime edges (JNI, Rust FFI, runtime traces) — v2.
- DEFR-02: Build-system dependency graphs (Bazel/Buck) — v2.
- Broad repo tests still have earlier-phase failures outside Phase 04 scope (see `phases/04-historical-code-intelligence/deferred-items.md`).

**Technical Debt Incurred:**

- Symbol-first adoption rate metric (scorecard) requires future per-op granularity in telemetry events before it can be automatically computed.
- Bootstrap cost benchmark needs automated threshold before it becomes a hard gate.
