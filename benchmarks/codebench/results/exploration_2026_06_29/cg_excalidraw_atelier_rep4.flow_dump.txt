## Data model → render

**Source of truth: `Scene`** (`packages/element/src/Scene.ts`). Holds `elements` + derived caches: `nonDeletedElements`, `nonDeletedElementsMap`, frames. Every element carries a `version` + `versionNonce`.

**Two canvases, layered** — Excalidraw splits work across separate `<canvas>`:
- **StaticCanvas** — committed elements (`renderStaticScene`, `renderer/staticScene.ts`)
- **InteractiveCanvas** — selection, handles, cursors (`renderer/interactiveScene.ts`)
- plus `NewElementCanvas` for the element being drawn (optimization — drawn once on top instead of re-rasterizing the whole static scene each pointermove).

Separation means dragging a selection box only repaints the cheap interactive layer, not every committed element.

## Update → repaint flow

1. **Mutate.** `Scene.mutateElement(el, updates)` → `mutateElement` bumps `version`/`versionNonce` if anything actually changed. Only if version changed AND `informMutation` → `triggerUpdate()`. (`replaceAllElements` for bulk swaps rebuilds the maps, then `triggerUpdate`.)
2. **`triggerUpdate()`** — `sceneNonce = randomInteger()` + fires all `onUpdate` callbacks.
3. **React bridge.** `App` registers `this.scene.onUpdate(this.triggerRender)` (App.tsx:3147). `triggerRender()` calls `this.setState({})` — empty setState just to force a re-render (`force` variant instead calls `scene.triggerUpdate()` to also invalidate the canvas nonce).
4. **`App.render()`** → `this.renderer.getRenderableElements(...)` (`scene/Renderer.ts`). Builds the renderable `elementsMap` (drops the currently-edited text + framed new-element), computes viewport-visible elements via `isElementInViewport`. **Memoized** (`memoize` on a `canvasNonce = sceneNonce[:newElement.versionNonce]`) — same nonce + same viewport params → returns cached list, no recompute.
5. **StaticCanvas** (`React.memo`, custom `areEqual` shallow-comparing `elementsMap`/`renderConfig`) receives `elementsMap`+`visibleElements` as props; a `useEffect` calls `renderStaticScene(...)`.
6. **`renderStaticSceneThrottled = throttleRAF(...)`** — coalesces all paints to one `requestAnimationFrame`; rapid mutations in a frame collapse to a single draw (keeps only `lastArgs`).
7. **Draw.** `renderStaticScene` iterates `visibleElements` → `renderElement(...)` per element.

## Per-element caching

`renderElement` rasterizes each element to an offscreen canvas cached in a `WeakMap` keyed by the element object (`elementWithCanvasCache`, renderElement.ts:603). Reused unless invalidated by: zoom change, theme, bound-text version, image crop, containing-frame opacity, or arrow-angle-vs-label. Otherwise the cached bitmap is just blitted (`drawElementFromCanvas`). WeakMap keying means a new element version (mutation produces logically-new object) misses cache → regenerates; unchanged elements blit for free.

## Net effect
Nonce-gated memoization (skip recompute) → RAF throttle (coalesce paints) → per-element WeakMap bitmap cache (skip re-rasterize) → static/interactive layer split (repaint only what moved). A mutation touches one element's version; everything downstream reuses caches for the rest.
