# Phase 19 Plan 01 — Expanded SCIP registry discovery

## Goal

Expand SCIP binary discovery to all M4-supported canonical languages while preserving legacy env-var contracts.

## Tasks

1. Add bare fallback `scip_indexer` names in `languages.py` for Go, Rust, Java, Ruby, C, and C++.
2. Introduce explicit SCIP binary specs in `binaries.py`.
3. Preserve `ATELIER_SCIP_PYTHON_BIN` and `ATELIER_SCIP_TYPESCRIPT_BIN` byte-identically.
4. Add env vars for Go, Rust, Java, Ruby, and Clang.
5. Generalize `discover_scip_binaries()` to iterate supported specs.
6. Add tests for env override precedence, fallback discovery, Rust subcommand metadata, and C/C++ shared Clang env var.
