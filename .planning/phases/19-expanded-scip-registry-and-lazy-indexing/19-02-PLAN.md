# Phase 19 Plan 02 — Lazy SCIP artifact indexing

## Goal

Add an explicit, opt-in SCIP indexer runner that writes repo-local `.scip` artifacts without automatic startup indexing.

## Dependencies

- Phase 19 Plan 01 must complete first.

## Tasks

1. Add a structured indexing result type in `indexer.py`.
2. Add `ScipIndexer.index_language(language, timeout_seconds=...)`.
3. Build argv from SCIP binary specs and run subprocesses with:
   - `shell=False`
   - pinned `cwd=repo_root`
   - captured output
   - explicit timeout
4. Normalize generated output to `cache_root / f"{language}.scip"`.
5. Skip cleanly when a binary is missing or required repo context is absent.
6. Add mocked runner tests for success, missing binary, missing C/C++ context, subprocess failure, timeout, and artifact rediscovery.
