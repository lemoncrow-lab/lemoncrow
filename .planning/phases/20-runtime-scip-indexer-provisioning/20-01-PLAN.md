# Phase 20 Plan 01 — Managed Tier-1 install and discovery

## Goal

Install cheap SCIP indexers into Atelier-managed runtime directories and make discovery prefer those directories before system `PATH`.

## Tasks

1. Update `scripts/install.sh` to include `scip-python` and `scip-typescript` in the managed npm global install list.
2. Add managed binary directory helpers in `binaries.py`.
3. Search env override first, then managed dirs, then system `PATH`.
4. Add tests for managed-dir precedence and Tier-1 package metadata.

- `uv run pytest tests/infra/code_intel/scip -q`
