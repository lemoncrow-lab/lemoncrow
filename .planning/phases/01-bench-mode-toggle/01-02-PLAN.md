<objective>
Add a `bench.is_off()` guard to each of the four capability entry points. When bench
mode is OFF, every capability short-circuits and returns a valid, zero-work passthrough
result instead of running its full logic — no LLM routing, no context dropping, no
memory reads, and no MCP tool substitution (the last is handled in Plan 01).

Purpose: These guards are the mechanism that makes ATELIER_BENCH_MODE=off produce a
clean baseline. Each guard is a single two-line addition at the top of the relevant
method. No business logic changes.

Output: Five files modified with minimal guards; a new classmethod on CompressionResult.
</objective>

def _load(self) -> list[MemoryFact]:

        for adapter in self._adapters:

        self._cache = facts
    return self._cache
