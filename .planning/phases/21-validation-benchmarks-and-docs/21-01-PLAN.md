# Phase 21 Plan 01 — Fixture Matrix and SCIP Availability

## Goal

Add automated validation for the canonical language matrix and the SCIP availability matrix.

## Tasks

1. Add representative fixtures under `tests/fixtures/languages/` for every canonical language.
2. Add `tests/core/test_language_matrix.py`:
   - Assert `language_for_path()` and `SemanticFileMemoryCapability.smart_read()` detect each language.
   - Assert tree-sitter-supported code/data fixtures return tree-sitter outlines when the 25% guard permits.
   - Assert tree-sitter-supported fixtures expose non-empty definition tags where definitions are expected.
3. Add `tests/infra/code_intel/scip/test_scip_availability_report.py`:
   - Assert availability statuses cover every SCIP spec language.
   - Assert Tier-1, Tier-2, and Tier-3 statuses/hints match bootstrap metadata.

## Verification

- `uv run ruff check tests/core/test_language_matrix.py tests/infra/code_intel/scip/test_scip_availability_report.py`
