┼ WHY — FOUR MOVES, ONE TIGHTER LOOP

01 Read smarter

code_search replaces grep loops — symbols, source, callers, callees, and blast radius in one call. read returns exact ranges, not whole files.

02 Think sharper

Narrow modes route work through the right shape — explore, plan, execute, solve — instead of one vague do-everything agent.

03 Talk less

Compact tool results, batched edits, telegraphic personas. The transcript holds decisions, not narration.

04 Never forget

Recall, session memory, and deduped reads stop the agent from paying twice for context it already found.
