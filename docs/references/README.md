# References

This directory maps the repository files that agents should treat as primary references.

## High-signal references

- `pyproject.toml` - Python dependencies and tool config
- `Makefile` - canonical validation commands
- `docker-compose.yml` - local stack wiring
- `frontend/package.json` - frontend scripts
- `scripts/install_*.sh` - host install behavior
- `src/atelier/gateway/hosts/configs/*.yaml` - host prompt template metadata
- `src/atelier/core/service/api.py` and `schemas.py` - service contract surfaces
